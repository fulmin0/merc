# Sourcing Standards

## Principle

Cite what you actually read. Never cite a source to add credibility if you only read a summary of it. The reader needs to know exactly what evidence each claim rests on.

## Evidence Tiers

| Tier | Type | Example |
|------|------|---------|
| 1 | RCTs and peer-reviewed studies | Solanto et al. (2010) MCT trial |
| 2 | Practitioner frameworks — primary text read directly | Dawson & Guare book, read cover to cover |
| 3 | Consensus papers, meta-analyses, review articles | Systematic review of EF interventions |
| 4 | Secondary summaries — publisher pages, blog posts, book descriptions | Guilford Press page, practitioner blog |

## Rules

- Every wiki page must declare its `evidence-tier` in frontmatter
- If claims come from multiple tiers, use the lowest tier present
- If a primary source is cited but not read, say so explicitly and mark Tier 4
- A page can be upgraded when the primary source is actually read

## "Not Yet Read" Flag

If a page cites books or papers not directly consulted, include this note in the sources section:

> **Not yet read:** [source]. Claims should be verified against primary text before being treated as authoritative.

## What This Is Not

This is a personal research wiki, not an academic paper. Formal citation styles are not required. What matters: can the reader find the source, and do they know what we actually looked at?
