# Overview: EF & BDEFS Research

## What This Wiki Is
A persistent knowledge base for understanding executive function deficits (specifically BDEFS) and building personal countermeasures. Maintained by Merc. You read it; Merc writes it.

## What We Know (So Far)

### The Map
Executive function = the control layer for goal-directed behavior. Not one thing — a family of separable but correlated capacities.

Two frameworks to hold simultaneously:
- **Miyake's 3 core functions:** Inhibition, Working Memory, Shifting
- **Barkley's 5 daily-life domains:** Time management, Organization/problem-solving, Self-restraint, Self-motivation, Emotional regulation

BDEFS is the assessment tool that operationalizes Barkley's model for real-world use.

### What Deficits Look Like
Procrastination. Missed deadlines. Lost things. Mental paralysis. Emotional flare-ups. Inconsistent follow-through. Not character flaws — neural architecture.

### What We Can Do About It
Five broad categories of interventions exist (physical activity, mindfulness, CBT, organizational training, external scaffolding). Most evidence is for children. Adult evidence is thinner but growing.

No single intervention is universal. Matching to individual deficit profile matters.

## The Stack
*Synthesis/inference — not a sourced finding. Logical integration of session 1 research.*

Four intervention types, four different roles. They're not alternatives — they're layers.

- **Scaffolding** routes around deficits. Immediate. No capacity change required to use it.
- **Exercise** builds the substrate. The only intervention with structural brain changes. Takes time and habit to deploy.
- **Medication** removes a chemical bottleneck. State-dependent — works while active, reverts when stopped.
- **MCT** teaches you to build and maintain scaffolding systematically. Adds cognitive restructuring for beliefs that block execution.

Order matters. Scaffold first — lowest activation energy, works immediately. Exercise next — needs enough functional capacity to build the habit. Medication and MCT are parallel layers depending on what's available and what's failing.

## So What
Start with scaffolding. One planner. All tasks in it. Specific times, not vague intentions. That's it to start — no new habits required beyond that one. Once there's enough functional capacity to build routines, add exercise — it's the only thing that actually changes the hardware. Medication removes a real bottleneck if it's on the table; not cheating. If self-directed scaffolding isn't sticking after a genuine attempt, Solanto's MCT is the most evidence-backed 12-week structure that exists. Pick layers based on severity and which BDEFS domains hit hardest.
